fabfile.g8
================================================================================


概要
------------------------------------------------------------

個人利用fabfile集です。

[giter8](https://github.com/n8han/giter8)用リポジトリになっており、  
ブランチを切り替えることで様々な用途に向けたfabfileを構成できます。


ブランチリスト
------------------------------------------------------------

### vagrant-single-mysql

ローカルでのVagrantを使って、MySQLのみ1台構成のVMを構築します。

```
$ g8 nisshiee/fabfile -b vagrant-single-mysql
```
